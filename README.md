##  任务定义与应用背景
### 任务定义

构建一个面向计算机学习者的多模态智能辅导Agent，支持：

1.按学科分类提问，如数据结构、计算机组成原理、操作系统、计算机网络、人工智能、软件工程等

2.多模态输入：文字、代码片段、截图（如算法流程图、计组电路图）、PDF/PPT讲义、项目目录压缩包

3.多模态输出：自然语言解释、代码注释、知识图解等

4.自动将解答内容提炼为知识点，存入对应分类的知识库，供复习查阅

### 应用背景

· 计算机学科理论知识错综庞杂，计算机课程内容抽象

· 现有大模型对于计算机学科相关问题理解能力和解答能力欠缺

· 现有大模型缺少学科知识结构化积累能力与总结能力
